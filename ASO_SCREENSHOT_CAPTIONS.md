## Design Rules

1. Caption text goes at the **TOP** of each screenshot (OCR reads top-down)
2. Use **bold headline** + smaller keyword-rich subtext
3. Font must be large enough to be OCR-readable (min 40pt equivalent)
4. Background should have good contrast with text
5. Never use text over busy gameplay screenshots — use clean backgrounds
6. Each locale gets its own set — do NOT use English text in non-English locales
